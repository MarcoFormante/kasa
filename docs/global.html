Loading() → {JSX.Element}

Route Segment Loading UI

This is a specialized Next.js component that acts as a streaming boundary. It is automatically rendered by the framework while a route segment is being fetched or during data-heavy server transitions.

Key Features:

  • Instant Feedback: Prevents white screens by showing an immediate UI spinner.
  • Global Wrapper: Uses the specialized LoadingPage to maintain visual consistency.
  • Non-Blocking: Allows the user to interact with the Header and Navigation while content loads.

RootLayout(props) → {JSX.Element}

Main Application Layout

The core structural component that wraps every page in the application. It defines the HTML skeleton and provides global services to all routes.

Responsibilities:

  • Typography: Injects the Inter font into the HTML body.
  • Context Providers: Initializes FavoritesContext and AlertHandlerContext for global state management.
  • Navigation: Renders the common Header component across all routes.
